**Project Overview**

This project predicts whether a loan application will be approved or not using Machine Learning techniques.
The model learns patterns from past loan application data and predicts loan approval status.

**Steps Performed**
-Data cleaning and preprocessing
-Exploratory Data Analysis (EDA)
-Feature selection
-Model training
-Model evaluation

**Technologies Used**
-Python
-Pandas
-NumPy
-Matplotlib
-Seaborn
-Scikit-learn
-Jupyter Notebook
